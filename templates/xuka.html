ĐÁNH GIÁ PHẦN MỀM XUKA 4.0

1. Tổng quan

  • Tên phần mềm: XUKA 4.0
  • Chức năng chính:
    • Đăng nhập bằng mã QR hoặc tài khoản
    • Thi trắc nghiệm theo mã đề
    • Tự động chấm điểm, hiển thị kết quả
    • Tải kết quả về máy, lưu lên server
    • Ngăn gian lận thi cử
  • Đối tượng sử dụng: Học sinh THPT – Giáo viên
  • Công nghệ: HTML, CSS (Tailwind), JavaScript, html5-qrcode, backend API
  • Tương thích: Chrome, Edge, Firefox (máy tính)
